## Run Application

1. Install dependencies

```
npm install
```

2. Create a local `.env` file (copy from `.env.example`) and set your local PostgreSQL values.

Format:
PGHOST=localhost
PGPORT=5432
PGDATABASE=auth
PGUSER=postgres
PGPASSWORD=your_postgres_password

3. Set up / normalize database tables

```
npm run db:setup
```

4. Start the app

```
npm run dev
```
